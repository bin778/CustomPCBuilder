## 프로젝트 개요
컴퓨터 부품 및 지식이 부족한 사람들을 위해 원하는 용도와 가격으로 조립 컴퓨터를 구매할 수 있는 CustomPCBuilder입니다.

## 개발 기간
* 2023.08.01~2023.12.15

## 개발 환경
#### Front-End
* `React.js`
* VSCode(HTML5, SCSS, JavaScript)
#### Back-End
* `Node.js`
* Django(Python)
* **Database**: MySQL 8.0

## 멤버
* 최영빈(개인 졸업 프로젝트)
